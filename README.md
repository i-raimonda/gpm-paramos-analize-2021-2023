# GPM paramos analizė (2021–2023)

Power BI analitinis projektas, skirtas GPM paramos duomenų analizei 2021–2023 m. laikotarpiu.

## Analizės apimtis
Ataskaitoje nagrinėjama:
- GPM paramos sumų dinamika
- paramos pasiskirstymas pagal gavėjus ir kategorijas
- paramos koncentracija ir pagrindinės tendencijos

## Duomenys
Analizei naudojami vieši, agreguoti duomenys, integruoti Power BI ataskaitoje.

## Technologijos
- Power BI Desktop
- DAX
- Vienos lentelės duomenų modelis
